# Purpose

This repository contains a set of small projects made as part of course "Techniki inteligentnej analizy danych" (TIAD - Techniques of intelligent data analysis) taken as part of "informatyka stosowana" (applied computer science) at Technical University of Lodz.

# Contents

Below you can find short descriptions of projects in this repository. For more detailed information, refer to `README.md` files in respective project folders.

### task-1

An application for converting table-like `.xlsx` files into `.docx` and `.pdf` files with two options for opinionated formating.
